## 구글 주가 데이터

```python
df = pd.read_excel('GOOG.xlsx')

```

```python

```

## KPSS 검정
```python
from statsmodels.tsa.stattools import kpss

```

## 차분

1차 차분
```python
yd = y.diff().dropna()
plot_acf(yd)
```

2차 차분: 차분한 값을 다시 차분
```python
yd2 = yd.diff().dropna()
plot_acf(yd2)
```

### 2차 차분한 결과의 KPSS 검정
```python
kpss(yd2)
```
