# 자세 추정

## 자세 추정
* 모델
```python
from ultralytics import YOLO
model = YOLO('yolo11n-pose.pt')
```
* 이미지 불러오기
```python
import cv2 as cv
img = cv.imread('bus_stop.jpg')
```
* 자세 추정
```python
results = model(img)
```
