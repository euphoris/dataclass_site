# 컴퓨터 비전과 딥러닝

인공지능 ⊃ 머신러닝  ⊃ 딥러닝

인공지능: 인공적으로 컴퓨터를 똑똑하게 

고전적 인공지능은 길찾기 알고리즘과 같은 것에 집중

그러나 이것만으로는 부족, 왜? 길찾기는 지도가 있다는 전제 하에 가능하나, 현실에는 지도가 없음

머신러닝: 데이터 -> 패턴

 - 지도학습: x -> y
 - 비지도학습
 - 강화학습

## 딥러닝

뉴런은 자극이 약하면 반응하지 않고(0), 강하면 반응을 하는(1) 신경 세포

생물학적 신경망 영감을 받은 인공신경망

다층신경망: 여러 층으로 이루어진 신경망

딥러닝으로 발전

보편근사정리: 다층신경망이 충분히 크면 어떤 함수든지 근사할 수 있다

충분히 크면(sufficiently large): 어떤 크기 N이상에서는 특정 성질을 만족하는 그런 N이 존재한다. (다시 말해 N에서 만족하면 N+1, N+2, ...의 크기에서도 만족한다)

경사하강법: 오차가 줄어드는 방향(경사)으로 파라미터를 업데이트해서 오차를 줄임(하강)

전통적 ML과 달리 딥러닝에서는 end-to-end로 학습할 수 있다. why? 표상학습을 하기 때문.
