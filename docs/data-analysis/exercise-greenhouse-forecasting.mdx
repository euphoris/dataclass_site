# 종합 실습: 온실 가스 배출 예측

- 과거 온실가스 배출량의 추세와 주요 패턴(계절성 등)을 명확히 파악
- 향후 1~2년간의 월별 온실가스 배출량을 예측
- 감축 목표 달성 가능성을 점검하고 선제적인 관리 방안(예: 에너지 효율 개선 투자 시점 결정, 재생에너지 도입 확대 계획 수립 등)을 마련하고자 함

| 컬럼 | 설명 |
|---|---|
| date | 측정 날짜 |
| emission | 온실가스 배출량 |
| production | 생산량 |
| temperature | 기온 |

[실습 파일 green_house_gas.xlsx](https://github.com/euphoris/datasets/raw/refs/heads/master/green_house_gas.xlsx)

1. 이 데이터로 `emission ~ production + temperature` 모델을 만들어 OLS 회귀분석을 해보세요.
2. 3년 후 온실 가스 배출량을 예측해보세요. (주의: 이 데이터는 월 단위 데이터입니다.)
