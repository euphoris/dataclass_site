# 상호작용
## 상호작용 (Interaction)

*   **정의:** 한 독립변수의 효과가 다른 독립변수 **수준에 따라 달라지는** 경우.
*   **모형:** 두 독립변수 **곱으로 이뤄진 항(상호작용항)** 추가.
    $$ y = w_1 x + w_2 m + w_3 xm + b $$
*   **Python 관계식:**
    *   콜론(`:`): **상호작용항만** 추가 (`x:m`).
    *   별표(`*`): 각 변수 **주효과(main effect) 및 상호작용항 모두** 추가 (`x*m` 은 `x + m + x:m` 과 동일).

## 상호작용 예시 (x:연속형, m:범주형(0/1))

*   **상호작용 없는 경우 (`y ~ x + m`):**
    *   `m` 값 따라 **절편만** 바뀜 (기울기 동일). 두 직선 **평행**.
*   **기울기만 달라지는 경우 (`y ~ x + x:m`):**
    *   `m` 값 따라 **기울기만** 바뀜 (절편 동일). 두 직선 **원점(또는 특정 점) 교차**.
*   **기울기/절편 모두 달라지는 경우 (`y ~ x + m + x:m` 또는 `y ~ x*m`):**
    *   `m` 값 따라 **절편과 기울기 모두** 바뀜. 두 직선 **평행하지 않고 만나지도 않을 수 있음**.

## Python 상호작용 코드

```python
# 절편 다른 경우 (평행)
m1 = ols('y ~ x + m', data=d1).fit()

# 기울기 다른 경우 (교차)
m2 = ols('y ~ x + x:m', data=d2).fit() # 또는 ols('y ~ x*m', data=d2).fit() 사용 가능

# 절편/기울기 모두 다른 경우
m3 = ols('y ~ x + m + x:m', data=d3).fit() # 또는 ols('y ~ x*m', data=d3).fit() 사용 가능
```

## 반응표면분석법 (Response Surface Methodology, RSM)

*   **목적:** 여러 입력 변수(요인)와 출력 변수(반응) 간 관계 탐색/모델링하여, 반응 변수 **최적화 조건** 찾는 통계 기법.
*   **활용:** 공정/제품 성능 개선, 개발 시간 단축, 최적 운영 조건 탐색.
*   **특징:**
    *   **실험 계획법 (DOE)** 사용 (예: 중심합성설계 CCD, 박스-벤켄 설계 BBD).
    *   수학적 모델링 (주로 **2차항, 상호작용항 포함** 모델 사용).
    *   **곡면(비선형)** 관계 파악 및 **최적점** 탐색.
    *   곡률 모델링 위해 최소 3수준 이상 실험 필요.
