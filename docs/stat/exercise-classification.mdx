# 종합 실습: 재무 건전성 분류

- 생산 연속성과 회사 경쟁력 확보를 위해 안정적인 공급망 유지 필요 있음
- 협력 업체가 재무적 어려움을 겪을 경우 부품 및 원자재 공급에 문제가 생길 수 있음
- 협력 업체의 재무 상태를 감지하고 선제적 대응 필요
- 과거 협력 업체의 재무 데이터와 실제 부도/경영 악화 여부 데이터를 활용하여 신규 및 기존 협력 업체의 재무 건전성을 평가하고 잠재적 부도 위험을 예측하고자 함

[실습 데이터(supplier.xlsx)](https://github.com/euphoris/datasets/raw/refs/heads/master/supplier.xlsx)

위 공급 업체 데이터를 바탕으로 Random Forest 모형을 이용하여 `risk` 변수를 예측하는 머신러닝을 수행하고 다음 질문에 답하세요. (데이터는 8:2로 나누어 훈련 및 테스트 데이터셋을 구성합니다. `random_state=42`로 설정합니다.)
