# 기술 통계: 분위수



## 분위수 (Quantile)

*   데이터 크기 순 정렬 시, 전체 데이터를 특정 비율로 나누는 위치의 값.
*   **예:** 중간값 = 0.5 분위수 (50% 지점).
*   사분위수, 백분위수 등이 있음.

## 사분위수 (Quartile)

*   데이터를 **4등분**하는 위치 값.
*   **제1사분위수 (Q1):** 하위 25% 지점 (0.25 분위수).
*   **제2사분위수 (Q2):** 50% 지점 (0.5 분위수) = **중간값**.
*   **제3사분위수 (Q3):** 상위 25% 지점 (0.75 분위수).
*   (영어 철자 주의: quantile vs. quartile)

## 백분위수 (Percentile)

*   데이터에서 순위를 **백분율(%)**로 표현.
*   최소값 = 0 퍼센타일.
*   제1사분위수 (Q1) = 25 퍼센타일.
*   제2사분위수 (Q2) = 중간값 = 50 퍼센타일.
*   제3사분위수 (Q3) = 75 퍼센타일.
*   최대값 = 100 퍼센타일.

**최소값:**
```python
df.price.min()
```
**최대값:**
```python
df.price.max()
```
**25 퍼센타일 (제1사분위수):**
```python
df.price.quantile(0.25) # 인자로 비율(0~1) 입력
```
