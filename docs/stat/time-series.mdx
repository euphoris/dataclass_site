# 시계열 데이터 분석의 소개

## 시계열 (Time Series)

*   동일한 변수의 **시간에 따른 변동** 나타낸 데이터.
*   **예시:** 일별 고객 수, 월별 가격, 시간별 온도.

## 시계열 분석 종류

*   **예측 (Forecasting):** 미래 값이나 추세 예측.
*   **대치 (Imputation):** 센서 오류, 데이터 손상 등으로 누락된 데이터 보완.
*   **이상 탐지 (Anomaly Detection):** 비정상적 패턴 식별.
*   **분류 (Classification):** 시계열 특성 기반 레이블/범주 할당.

## 시계열 분석 사례

*   **예측:** 날씨 예측, 가격 예측, 수요 예측 (재고 관리).
*   **대치:** 센서 오류 데이터 보완, 데이터 손상 복원, 측정 부재 보완.
*   **이상 탐지:** 사이버 보안 트래픽 모니터링, 신용 카드 사기 탐지, 산업 기계 고장 예측.
*   **분류:** 음악 장르 분류, 사용자 활동 인식, 음성 감정 분석.

## 시계열 구성 요소

*   **추세 (Trend):** 데이터의 장기적 증가/감소 패턴 (예: 연간 치즈 생산량 증가).
*   **계절성 (Seasonality):** 달력 상 특정 시점(월, 분기, 요일 등)마다 반복되는 패턴 (예: 월별 음료 출고량).
*   **주기성 (Cycle):** 일정 빈도 아닌 형태의 장기적 증감 패턴. 보통 계절성보다 간격 김 (예: 6~10년 주기 주택 거래량 변동).
*   **자기 상관 (Autocorrelation):** 이전 값과 이후 값 사이 상관 관계.
    *   **양의 자기상관:** 이전 추세 유지 경향 (+면 +, -면 -).
    *   **음의 자기상관:** 이전 추세 반전 경향 (+면 -, -면 +).
*   **비정상 (Non-stationary):** 시계열 통계적 구조(평균, 분산 등) 시간에 따라 변하는 상태. 대부분 시계열 분석은 **정상(stationary)** 시계열 가정. (예: 추세 있는 주가 데이터는 비정상).
*   **이상점 (Outlier):** 일반적 패턴에서 벗어난 특이 사례 (예: 공장 화재로 인한 생산량 급감).
