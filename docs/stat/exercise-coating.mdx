# 종합 실습: 도장 품질 예측

- 외장 패널의 도장 품질은 도막 두께 (coating thickness) 와 도막 밀착력 (adhesion strength) 가 관리 지표
    - 도막 두께는 균일하게 목표 범위(예: 95µm ~ 105µm) 내
    - 밀착력은 가능한 높아야(예: 최소 5 MPa 이상)
- 공정에서 건조 온도 (drying temperature), 라인 속도 (line speed), 스프레이 압력 (spray pressure)을 변경할 수 있음
- 최적의 도막 두께와 밀착력을 동시에 만족시키는 파라미터를 찾아야함

|컬럼|설명|
|---|---|
|temperature|건조 온도(°C)|
|line_speed|라인 속도(m/min)|
|spray_pressure|스프레이 압력(bar)|
|thickness|도막 두께(µm)|
|adhesion|밀착력(MPa)|

도막 두께와 밀착력은 다음과 같은 수식을 따를 것으로 추측됨

$$
\text{temperature}^2 + \text{line\_speed}^2 + \text{spray\_pressure}^2 + \text{temperature} + \text{line\_speed} + \text{spray\_pressure}
$$


위의 수식을 이용하여 도막 두께를 예측하는 모형 `thickness_model`과 밀착력을 예측하는 모형 `adhesion_model`을 만들어보세요. 그리고 이를 다음의 실험 데이터에 적용하여 계수를 추정하세요.
