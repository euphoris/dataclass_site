# 1종 오류와 2종 오류


|             | 가설검정: 귀무가설 기각 | 가설검정: 귀무가설 기각 실패 |
| :---------- | :-------------------- | :----------------------- |
| **실제: 참** | **1종 오류**<br/>False Alarm<br/> | **적중**<br/>Hit |
| **실제: 거짓**| **정확 기각**<br/>Correct Rejection | **2종 오류**<br/>Miss<br/> |


## 1종 오류 (Type I Error)

- 정의
    - "실제로는 차이가 없는데, 차이가 있다고 잘못 결론 내리는 오류"
    - 즉, 귀무가설(H₀: 차이가 없다)이 사실인데도 이를 기각하는 오류
    - False Alarm

- $\alpha$ (알파, 유의수준)
    - 1종 오류를 범할 최대 허용 확률
    - 우리가 보통 유의수준을 $\alpha = 0.05$로 정하는 것은, "1종 오류를 범할 위험을 5%까지는 감수하겠다"는 의미
    - 예: "새 공정이 기존 공정과 성능이 같은데도, 더 좋다고 잘못 판단하여 불필요한 공정 교체 비용을 발생시키는 상황"

## 2종 오류 (Type II Error)
*   2종 오류 (Type II Error)
    *   "실제로 차이가 있는데, 차이가 없다고 잘못 결론 내리는 오류"
    *   귀무가설이 거짓인데도 이를 기각하지 못하는 오류
    *   Miss
*   예: "획기적으로 성능이 좋은 신소재를 개발했지만, 실험에서 차이를 발견하지 못하고 개발을 포기하게 되는 상황"
*   $\beta$ (베타): 2종 오류를 범할 확률
*   검정력 (Power) = $1 - \beta$
    *   "실제로 차이가 있을 때, 그것을 올바르게 '차이가 있다'고 탐지해낼 확률"
    *   일반적으로 80% 또는 90% 이상의 검정력을 목표
